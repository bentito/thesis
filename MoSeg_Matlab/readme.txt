MoSeg_Matlab 
===============

Richard Lange
richard.d.lange.13@dartmouth.edu

This software was developed at Dartmouth College as part of undergraduate thesis work by Richard Lange.
The primary development environment was MATLAB R2011b 64-bit Mac. It has also been tested on MATLAB R2010b 64-bit Linux.


Introduction
------------

This is an implementation of the Motion Segmentation given in the following paper:

    T. Brox, J. Malik: Object Segmentation by Long Term Analysis of 
    Point Trajectories, Proc. European Conference on Computer Vision, 2010.
